# pet diary
Creating an api with nodejs.


Routes:
POST - posts/create - create a post 
GET - posts/:id - bring a list of posts by Id
POST - users/create - create a user
POST - users/auth - login a user
GET - users/ - list total users

# play the project
node app.js
